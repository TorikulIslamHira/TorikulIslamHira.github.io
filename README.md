## Getting Started

No installation or build step is needed.

1. **Clone the repository**
   ```bash
   git clone https://github.com/TorikulIslamHira/TorikulIslamHira.github.io.git
   cd TorikulIslamHira.github.io
   ```

2. **Open locally**
   - Open `index.html` directly in any modern browser, **or**
   - Use a simple local server for the best experience:
     ```bash
     # Python 3
     python -m http.server 8080
     # Then visit http://localhost:8080
     ```
